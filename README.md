# Private Pub Quiz

A real-time, mobile-first pub quiz application for playing with friends. Features team-based scoring, speed bonuses, and a multi-lobby system.

## Tech Stack
- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Socket.IO
- **Infrastructure:** Docker Compose
